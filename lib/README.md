# Lib Directory

Utility functions, API clients, and helper libraries.

## Structure

- `api-client.ts` — Axios HTTP client with interceptors
- `supabase-client.ts` — Supabase JS client
- `apify-client.ts` — Apify API wrapper
- `gemini-client.ts` — Google Gemini API wrapper
- `auth.ts` — NextAuth configuration
- `redis.ts` — Redis/Bull client
- `db.ts` — Supabase types and utilities
- `validators.ts` — Zod input validation
- `utils.ts` — Helper functions
