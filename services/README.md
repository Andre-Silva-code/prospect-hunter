# Services Directory

Business logic and integration services.

## Structure

- `prospect.service.ts` — Prospect operations (CRUD, filtering, scoring)
- `message.service.ts` — Message operations (create, approve, send)
- `metrics.service.ts` — Calculate KPIs and analytics
- `apify.service.ts` — Apify integration logic
- `gemini.service.ts` — Gemini API integration logic
- `queue.service.ts` — Bull job queue management
- `auth.service.ts` — Authentication helpers
