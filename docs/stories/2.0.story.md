# Story 2.0: Métricas Operacionais e Fila de Follow-up

## Story Statement

> **As a** dono de agência de tráfego pago,
> **I want** ver métricas rápidas e a fila dos próximos follow-ups,
> **So that** eu saiba onde agir primeiro todos os dias.

## Acceptance Criteria

1. [x] O CRM exibe métricas de leads totais, pendentes, enviados e respondidos.
2. [x] O CRM calcula taxa de resposta.
3. [x] Existe fila de follow-up priorizada por score e status.
4. [x] Leads que já responderam saem da fila.
5. [x] Os filtros atuais impactam a fila operacional.
6. [x] Existe teste cobrindo métricas e fila.
7. [x] Checklist e file list foram atualizados.

## Tasks / Subtasks

- [x] Derivar métricas operacionais a partir dos leads.
- [x] Exibir cards de métricas no topo do workspace.
- [x] Construir fila de follow-up priorizada.
- [x] Exibir fila com contexto de score, etapa e status.
- [x] Adicionar teste de métricas e fila.
- [x] Validar com `npm run lint`, `npm run typecheck` e `npm test`.
