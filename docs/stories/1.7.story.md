# Story 1.7: Busca, Filtros e Score Automático no Funil

## Story Statement

> **As a** dono de agência de tráfego pago,
> **I want** priorizar leads por score e encontrar rapidamente contatos no funil,
> **So that** a prospecção diária foque nas melhores oportunidades primeiro.

## Acceptance Criteria

1. [x] Cada lead recebe score automático no cadastro.
2. [x] Cada lead recebe prioridade derivada do score.
3. [x] Existe busca textual por empresa, nicho, região, contato ou gatilho.
4. [x] Existe filtro por prioridade.
5. [x] O funil reflete os filtros sem quebrar a movimentação de etapa.
6. [x] Existe cobertura de teste para score visível e filtros.
7. [x] Checklist e file list foram atualizados.

## Tasks / Subtasks

- [x] Estender tipos para score e prioridade.
- [x] Implementar cálculo automático de score no cadastro.
- [x] Adicionar controles de busca e filtro.
- [x] Exibir score/prioridade nos cards do funil.
- [x] Adicionar teste para filtros e score.
- [x] Validar com `npm run lint`, `npm run typecheck` e `npm test`.
