# Story 1.9: Edição, Cópia e Status Comercial da Mensagem

## Story Statement

> **As a** dono de agência de tráfego pago,
> **I want** editar, copiar e registrar o resultado da mensagem enviada,
> **So that** o CRM reflita contato real com cada lead.

## Acceptance Criteria

1. [x] Cada lead possui status comercial visível.
2. [x] A mensagem sugerida pode ser editada antes do envio.
3. [x] Existe ação para copiar a mensagem.
4. [x] Existe ação para marcar `Mensagem enviada`.
5. [x] Existe ação para marcar `Respondeu`.
6. [x] A persistência local armazena os novos campos.
7. [x] Existe cobertura de teste para edição, cópia e atualização de status.
8. [x] Checklist e file list foram atualizados.
