# Story 1.8: Geração de Mensagens de Prospecção por Lead

## Story Statement

> **As a** dono de agência de tráfego pago,
> **I want** gerar uma mensagem inicial contextual para cada lead,
> **So that** eu consiga abordar rápido sem escrever tudo do zero.

## Acceptance Criteria

1. [x] Cada lead pode abrir uma mensagem sugerida dentro do card.
2. [x] A mensagem considera empresa, nicho, região, prioridade e gatilho.
3. [x] A lógica de geração fica separada do componente visual.
4. [x] A persistência local continua válida com os campos de score e prioridade.
5. [x] Existe cobertura de teste para a geração da mensagem.
6. [x] Checklist e file list foram atualizados.
